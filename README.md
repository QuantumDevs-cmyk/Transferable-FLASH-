The platform supports popular cryptocurrencies such as USDT, BTC, USDC, and DAI while maintaining compatibility with leading crypto wallets and blockchain ecosystems.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/transferable-flash-usdt.git
```

Move into the project directory:

```bash
cd transferable-flash-usdt
```

Install requirements:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
